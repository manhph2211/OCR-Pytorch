## 2.Usage :smiley:

- First, `git clone https://github.com/manhph2211/OCR-Pytorch.git`
- Get all the dependencies: `pip3 install -r requirements.txt`
- Dataset : `unzip captcha_images_v2.zip`
- Train : `python3 train.py`
